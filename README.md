# Installation :

- Cloner le répertoire : ``git clone https://github.com/xReapex/angular_csid.git``.
- Installer les dépendances : ``npm install``.
- Lancer le serveur : ``ng serve``.
- Ouvrir l'url du [serveur](http://localhost:4200/).
